# 📱 COD TP - دليل الاستخدام الكامل

## 🎯 مقدمة

**COD TP** هو تطبيق شامل لتتبع وإدارة الطلبات للتجار الإلكترونيين في الجزائر بنظام الدفع عند الاستلام.

---

## 🚀 البدء السريع

### 1️⃣ فتح التطبيق
- افتح ملف `index.html` في متصفحك
- أو قم بتحميل `mobile.html` على هاتفك

### 2️⃣ إنشاء حساب
```
- انقر على "إنشاء حساب جديد"
- أدخل بيانات متجرك
- ستحصل على حساب فوراً
```

### 3️⃣ الدخول للتطبيق
```
- البريد الإلكتروني: بريدك المسجل
- كلمة المرور: كلمتك المسجلة
```

## 🎮 الميزات الرئيسية

### 1. 📊 لوحة التحكم
- عرض إحصائيات شاملة عن الطلبات
- معدل الإيصال والإلغاء
- الإيرادات الإجمالية

### 2. 📦 إدارة الطلبات

#### إضافة طلب جديد:
```
1. انقر على "+ طلب جديد"
2. أدخل بيانات العميل
3. حدد الوكيل (اختياري)
4. انقر "تأكيد"
```

#### تحديث حالة الطلب:
```
الحالات المتاحة:
- قيد الانتظار (Pending)
- مؤكدة (Confirmed)
- مُرسلة (Shipped)
- موصلة (Delivered) ✓
- ملغاة (Cancelled) ✗
```

#### عمليات الطلبات:
- ✏️ تعديل
- 🗑️ حذف
- 👤 إسناد لوكيل
- 📊 عرض التفاصيل

### 3. 👥 إدارة الوكلاء

#### إضافة وكيل جديد:
```
1. انقر على "+ وكيل جديد"
2. أدخل بيانات الوكيل:
   - الاسم
   - البريد الإلكتروني
   - الهاتف
   - كلمة المرور
   - المنطقة (اختياري)
   - الحد الأقصى للطلبات
3. انقر "إضافة"
```

#### مميزات الوكيل:
- حساب شخصي آمن
- طلبات مسندة حصرية
- إحصائيات فردية
- تطبيق هاتف مخصص

### 4. 🔗 ربط المتاجر الإلكترونية

#### المتاجر المدعومة:
- 🛍️ **Shopify**
- 📦 **YuKan**
- 🏪 **WooCommerce**
- 🛒 **PrestaShop**

#### خطوات الربط:
```
1. انقر على "+ ربط جديد"
2. اختر المتجر
3. أدخل بيانات الاتصال
4. أكمل المصادقة
5. ستتم مزامنة الطلبات تلقائياً
```

### 5. 📈 الإحصائيات والتقارير

#### المقاييس المتاحة:
- **إجمالي الطلبات**: عدد جميع الطلبات
- **معدل الإيصال**: نسبة الطلبات المسلمة
- **معدل الإلغاء**: نسبة الطلبات الملغاة
- **الإيرادات**: من الطلبات المسلمة فقط
- **أداء الوكلاء**: إحصائيات فردية لكل وكيل

### 6. 📱 تطبيق الهاتف (للوكلاء)

#### للعاملين لديك:
```
1. افتح mobile.html على الهاتف
2. سجل الدخول برقم البريد والمرور
3. شاهد طلباتك المسندة
4. حدّث حالة كل طلب
5. اعرض الإحصائيات الخاصة بك
```
