# COD TP - تطبيق تتبع الطلبات

تطبيق شامل لتتبع وإدارة الطلبات للتجار الإلكترونيين في الجزائر (الدفع عند الاستلام)

## المميزات الرئيسية

✅ **تتبع الطلبات** - من الإنشاء إلى الإيصال
✅ **إدارة الوكلاء** - حسابات متعددة مع صلاحيات مختلفة
✅ **تقسيم الطلبات** - يدويًا وتلقائيًا
✅ **إحصائيات ونسب مئوية** - طلبات موصلة وملغاة
✅ **ربط المتاجر** - Shopify, YuKan وغيرها
✅ **تطبيق الهاتف** - PWA متوافق مع جميع الأجهزة
✅ **API متقدمة** - للتكامل الكامل

## بنية المشروع

```
COD-TP/
├── index.html                 # الصفحة الرئيسية
├── dashboard.html             # لوحة التحكم الرئيسية
├── agents.html                # إدارة الوكلاء
├── orders.html                # إدارة الطلبات
├── integrations.html          # ربط المتاجر
├── statistics.html            # الإحصائيات والتقارير
├── mobile.html                # تطبيق الهاتف
├── css/
│   ├── styles.css            # الأنماط الأساسية
│   ├── dashboard.css          # أنماط لوحة التحكم
│   ├── responsive.css         # التجاوب مع الأجهزة
│   └── mobile.css             # أنماط الهاتف
├── js/
│   ├── app.js                # تطبيق الويب الرئيسي
│   ├── database.js            # إدارة قاعدة البيانات
│   ├── agents.js              # إدارة الوكلاء
│   ├── orders.js              # إدارة الطلبات
│   ├── integrations.js        # ربط المتاجر
│   ├── statistics.js          # الإحصائيات
│   ├── mobile.js              # تطبيق الهاتف
│   ├── api.js                 # واجهات برمجية
│   └── utils.js               # دوال مساعدة
├── data/
│   └── database.json          # قاعدة البيانات
├── manifest.json              # ملف PWA
└── service-worker.js          # خدمة الخلفية

```

## البدء السريع

1. فتح `index.html` في المتصفح
2. إنشاء حساب جديد
3. إضافة الوكلاء
4. ربط المتاجر الإلكترونية
5. البدء في إدارة الطلبات

## التكنولوجيا المستخدمة

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage & IndexedDB
- PWA (Progressive Web App)

## المتطلبات

- متصفح حديث (Chrome, Firefox, Safari, Edge)
- Visual Studio Code (للتطوير)
- محرر JSON للبيانات
